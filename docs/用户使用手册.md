## 一、平台简介

### 1.1 产品定位

**EcoMind-AI 智慧环保云平台**是悦恩科技自主研发的新一代企业级环境监测数据管理平台。平台以"**轻硬件、重软件、AI驱动**"为设计理念，为排污企业提供一站式的环保数据采集、存储、分析、预警和智能诊断服务。

与传统的省市级环保数据监管中心相比，EcoMind-AI 专注于为企业提供**更智能、更主动、更易用**的环保数据管理体验，帮助企业从"被动应对监管"转变为"主动预防风险"。

### 1.2 适用场景

- 工业废水排放监测
- 废气排放在线监测
- 重金属污染物监控
- 污水处理设施运行监控
- 环保设备运维管理

### 1.3 支持的监测协议

平台完全兼容国家标准 **HJ 212-2017** 和 **HJ 212-2025** 环保数据传输协议，可直接对接市面上主流的数采仪设备，无需额外改造。

### 3.2 六大核心特色

#### 特色一：AI 时序数据智能预测

**传统痛点**：企业只能看到历史数据和当前数据，无法预知未来趋势，往往在超标发生后才被动应对。

**EcoMind 方案**：

平台集成先进的时序预测算法，基于历史监测数据自动学习污染物变化规律，预测未来数据走势。

- **预测算法**: 采用 Prophet / NeuralProphet 神经网络预测模型
- **置信区间**: 提供 80% 置信区间可视化，直观展示预测不确定性
- **预警提示**: 当预测值接近阈值时提前预警，给企业充足的调整时间
- **可视化**: 历史数据实线 + 预测数据虚线 + 绿色置信区间带

**应用场景**：
- 预判 COD、氨氮等指标是否可能超标
- 合理安排污水处理设施的运行负荷
- 提前准备应急预案，避免环保事故

---

#### 特色二：悦恩 AI 智能诊断报告

**传统痛点**：监测数据只是冷冰冰的数字，企业环保人员缺乏专业知识解读数据背后的含义。

**EcoMind 方案**：

平台集成星火大模型，结合环保领域专业知识库，一键生成智能运维诊断报告。

- **综合分析**: 分析单台设备所有污染物数据，发现潜在问题
- **流式输出**: 实时生成报告，即时查看分析进度
- **专业建议**: 基于行业标准和最佳实践，提供运维优化建议
- **易于理解**: Markdown 格式化输出，层次清晰，重点突出

**诊断报告内容示例**：
```
## 设备运行状态诊断

### 一、数据质量评估
- 数据完整率: 98.5%
- 异常标志占比: 1.2%
- 评价: 数据质量良好

### 二、污染物趋势分析
- COD: 近24小时均值 45.2mg/L，呈下降趋势
- 氨氮: 近24小时均值 8.3mg/L，波动较小
- 总磷: 近24小时均值 0.45mg/L，接近预警值

### 三、风险提示
⚠️ 总磷浓度接近预警阈值(0.5mg/L)，建议关注...

### 四、运维建议
1. 检查除磷药剂投加量是否充足
2. 核查生化池运行参数
3. 建议增加采样频次监控
```

#### 特色三：多维度智能告警系统

**传统痛点**：传统系统只有简单的阈值超标告警，容易漏报或误报，告警信息淹没在大量通知中。

**EcoMind 方案**：

平台构建四维度告警体系，结合 AI 智能分析，精准识别真正需要关注的问题。

**告警类型**：

| 类型 | 说明 | 触发条件 |
|-----|------|---------|
| **阈值超标** | 监测值超过设定阈值 | 实测值 > 预警值/报警值 |
| **AI 异常检测** | AI 模型识别的异常模式 | 异常评分超过阈值 |
| **设备离线** | 设备失联告警 | 超过心跳超时时间无响应 |
| **标志异常** | 数采仪上报的异常标志 | D(无效)/T(超时)/F(故障)等 |

**告警分级**：

| 级别 | 颜色 | 说明 | 处理优先级 |
|-----|------|------|----------|
| 提示 | 蓝色 | 需要关注但不紧急 | 低 |
| 警告 | 橙色 | 可能演变为问题 | 中 |
| 严重 | 红色 | 需要立即处理 | 高 |

**智能去重**：30分钟内相同类型告警自动去重，避免告警风暴。

---

#### 特色四：短信实时通知 + 跟进机制

**传统痛点**：平台内的告警通知容易被忽略，关键问题无法及时传达到责任人。

**EcoMind 方案**：

集成腾讯云短信服务，关键告警实时推送至企业联系人手机。

- **即时通知**: 严重告警触发后立即发送短信
- **12小时跟进**: 告警未处理超过12小时，自动发送跟进短信
- **信息完整**: 短信包含设备名称、告警类型、当前数值
- **防骚扰**: 智能去重，避免重复短信

**短信内容示例**：
```
【悦恩环保】设备"1#排放口"发生COD超标告警，
当前值 85.3mg/L，超过阈值 80mg/L，请立即处理。
```

```
【悦恩环保】设备"1#排放口"的COD超标告警已持续12小时，
当前值 82.1mg/L，请务必核实处理。
```

## 五、常见问题

### Q1: 设备显示离线怎么办？

**可能原因**：
1. 数采仪网络连接中断
2. 数采仪配置的服务器地址或端口错误
3. 数采仪「主通道」未启用

**解决方法**：
1. 检查数采仪网络连接状态
2. 确认服务器地址和端口配置正确
3. 确保「是否为主通道」设置为「是」
4. 重启数采仪后观察是否恢复

### Q2: 为什么没有收到告警短信？

**可能原因**：
1. 企业联系电话未配置或格式错误
2. 告警级别未达到短信通知阈值
3. 30分钟内已发送过相同告警短信

**解决方法**：
1. 检查企业信息中的联系电话是否正确
2. 确认告警级别为「严重」
3. 等待去重窗口过期后会重新发送

### Q3: AI 预测不准确怎么办？

**说明**：

3. 预测结果仅供参考，请结合实际情况判断
